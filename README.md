A lightweight, customizable accordion component for React with support for both single and multiple item expansion modes. Perfect for FAQs, collapsible menus, and content organization with smooth animations and full accessibility support.

## ‚ú® Features

- **Dual Selection Modes**: Toggle between single (only one item open) or multiple (multiple items open) selection
- **TypeScript Support**: Built with TypeScript for excellent type safety
- **Fully Customizable**: Style every aspect with CSS or CSS-in-JS solutions
- **Accessible**: Built with ARIA attributes and keyboard navigation support
- **Smooth Animations**: CSS transitions for opening/closing animations
- **Zero Dependencies**: Lightweight with no external dependencies
- **Mobile Friendly**: Responsive design and touch-friendly interactions
- **Easy Integration**: Simple API with intuitive props

## üîß Development

### Project Setup

```bash
# Clone the repository
git clone https://github.com/mcmay01/react-single-multi-selection-accordion.git

# Navigate to project directory
cd react-single-multi-selection-accordion

# Install dependencies
npm install

# Start development server
npm run dev
```

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

### Building for Distribution

```bash
# Build the component library
npm run build

# The built files will be in the dist/ directory
```
